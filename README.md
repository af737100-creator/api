# نظام فحص ثغرات MikroTik (ماسح مباشر)

نظام فحص ذكي لاكتشاف الثغرات الأمنية في أجهزة MikroTik. النظام يعمل بنظام "الفحص المباشر" ولا يقوم بحفظ أي بيانات أو نتائج (No-Persistence).

## الهيكلية
- **backend/**: يحتوي على محرك الفحص والـ API.
- **scanner_agent/**: الماسح الميداني للشبكات.
- **config/**: قواعد الأمان المستخدمة في الفحص.

## كيفية الاستخدام
1. قم برفع الـ Backend على منصة استضافة (مثل Render).
2. استخدم رابط الـ API للفحص المباشر:
   `GET /scan?ip=...&user=...&password=...`
